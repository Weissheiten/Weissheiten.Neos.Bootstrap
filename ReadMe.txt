About
------
The Weissheiten Bootstrap package should allow its users to get an easy introduction to typo3 NEOS with the popular bootstrap template.
Starting from the great TYPO3.NeosDemoTypo3Org Site package the amount of available content elements has been significantly extended and at some places altered.

A big thank you to the awesome NEOS core team that gave me insight needed to create many of these new nodes.

Including Bootstrap
--------------------
The Weissheiten Bootstrap package does NOT use the TYPO3.Twitter.Bootstrap package at the moment.
So you have to include either that package or Bootstrap itself yourself in your main templates.
